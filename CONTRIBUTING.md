# Contribution guide

Please follow the existing folder structure when contributing

``` 
project
│   README.md
│   file001.txt    
│
└───source_name
│   │   source.py
│   │   schema.yml
│   └───analysis
│   │   └───topic (file or folder)
│   │   └───other_topic (file or folder)
│   └───model
│       └───model_name 
│       └───other_model_name
```
